## 三種新聞稿類別 & 細項子分類

### 類別 1：綜合新聞 / 即時
語調：事實導向、倒金字塔、5W1H | 字數：600-800

| 大類 | 細項 |
|------|------|
| 重大公告 | 酒店開幕/揭幕、品牌進駐/啟動、重大投資/擴建、策略合作簽約 |
| 人事與組織 | 高層人事任命、主廚到任/異動、組織改組/新團隊 |
| 獎項與認證 | 米其林/餐飲獎項、服務品質獎、ESG/永續認證、設計/建築獎項 |
| 即時快訊 | 緊急公告、營運異動、價格調整、服務更新 |

### 類別 2：商業 / 職場 / 餐飲
語調：專業產業視角、商業價值 | 字數：800-1000

| 大類 | 細項 |
|------|------|
| 餐廳新訊 | 新菜單/季節菜單、新餐廳開幕、餐廳改裝/升級、限定套餐/聯名菜單 |
| 主廚與團隊 | 客座主廚活動、主廚合作/交流、餐飲總監專訪、廚藝課程/工作坊 |
| 商務與宴會 | 商務宴請方案、婚宴/喜宴專案、會議/活動場地、包廂/私人聚餐 |
| 酒類與品飲 | 品酒晚宴/Wine Dinner、威士忌品飲活動、調酒師活動、酒單更新 |
| 招募與職場 | 大型招募活動、校園徵才、實習計畫、員工培訓/認證 |
| 商業合作 | 品牌策略合作、異業結盟、會員方案/聯名卡、供應商/產地合作 |

## 技術架構

| 元件 | 技術 |
|------|------|
| 前端 | React + Vite + TypeScript + Tailwind CSS + ShadCN/UI |
| 後端 | Express.js + TypeScript |
| AI 引擎 | Claude API (@anthropic-ai/sdk) + SSE 串流 |
| 文件萃取 | mammoth (DOCX) + pdf-parse (PDF) |
| 儲存 | 檔案式 JSON (無資料庫) |

### Prompt 組裝流程
1. **品牌標準層** — 品牌核心、個性、哲學、酒店資訊、餐廳/酒吧定位
2. **類別指引層** — 依選擇的類別載入對應的寫作風格指引
3. **Few-shot 範例層** — 從知識庫選出 2-3 篇最相關的歷史新聞稿
4. **使用者輸入層** — 主題、關鍵事實、日期、價格、地點、媒體聯繫人等
5. **輸出格式層** — 標題格式、引用格式、結尾格式等
